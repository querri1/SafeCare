# 安心宝 - 心理健康管理微信小程序

基于HTML设计图开发的微信小程序，提供心理健康管理功能。

## 功能特性

### 治愈中心
- 八段锦练习
- 五音疗法
- 心理健康自测
- PHQ-9 抑郁筛查
- GAD-7 焦虑筛查

### 数据中心
- 连续打卡统计
- 本周完成情况
- 本月成就展示
- 打卡记录图表
- 测试记录查看

### 个人中心
- 用户信息展示
- 我的数据
- 成就徽章
- 打卡日历
- 提醒设置
- 隐私设置
- 帮助与反馈

## 项目结构

```
miniprogram/
├── app.json          # 小程序配置
├── app.ts            # 小程序入口
├── app.scss          # 全局样式
├── components/       # 组件目录
│   └── navigation-bar/  # 导航栏组件
├── pages/            # 页面目录
│   ├── healing-center/  # 治愈中心页面
│   ├── data-center/     # 数据中心页面
│   └── profile/         # 个人中心页面
├── images/           # 图片资源
└── utils/            # 工具函数
```

## 技术栈

- 微信小程序原生开发
- TypeScript
- SCSS
- 自定义组件

## 开发说明

1. 使用微信开发者工具打开项目
2. 配置小程序AppID
3. 编译运行

## 页面说明

### 治愈中心 (healing-center)
- 展示今日治愈概览
- 提供各种治愈活动入口
- 显示完成状态和进度

### 数据中心 (data-center)
- 展示用户数据统计
- 提供图表可视化
- 显示测试记录

### 个人中心 (profile)
- 用户信息管理
- 功能菜单导航
- 设置和帮助入口

## 样式设计

- 采用现代化的卡片式设计
- 使用渐变色和圆角元素
- 响应式布局适配不同屏幕
- 统一的色彩体系 (#6366F1 主色调)

## 注意事项

- 需要配置tabBar图标文件
- 部分功能需要后端接口支持
- 建议在真机上测试体验效果
