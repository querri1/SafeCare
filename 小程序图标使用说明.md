# 安心宝小程序图标使用说明

## 图标文件

已生成符合安心宝小程序风格的图标文件：
- **SVG 源文件**：`miniprogram/images/app-icon.svg`
- **预览文件**：`miniprogram/images/app-icon-preview.html`

## 设计说明

### 设计理念
- **主题**：心形 + 盾牌，体现"安心守护"和"心理健康管理"
- **主色调**：渐变紫色 (#6366F1 → #8B5CF6)，与小程序整体风格一致
- **风格**：现代简洁，圆角设计，符合卡片式UI风格

### 设计元素
1. **主心形**：代表心理健康和关爱
2. **内部装饰心形**：使用渐变，增加层次感
3. **保护盾牌轮廓**：体现"安心"和"守护"的概念
4. **装饰圆点**：代表数据记录和统计

## 使用方法

### 方法一：使用在线工具转换（推荐）

1. 打开 SVG 文件：`miniprogram/images/app-icon.svg`
2. 使用在线 SVG 转 PNG 工具：
   - https://svgtopng.com/
   - https://convertio.co/zh/svg-png/
   - https://cloudconvert.com/svg-to-png
3. 上传 SVG 文件，选择输出尺寸：
   - **小程序图标**：1024×1024 像素
   - **TabBar 图标**：81×81 像素（可选）
4. 下载 PNG 文件

### 方法二：使用设计软件

1. 使用 Adobe Illustrator、Figma、Sketch 等设计软件打开 SVG 文件
2. 导出为 PNG 格式，尺寸选择：
   - **小程序图标**：1024×1024 像素
   - **TabBar 图标**：81×81 像素

### 方法三：使用预览页面

1. 在浏览器中打开 `app-icon-preview.html`
2. 右键点击图标，选择"另存为"
3. 或使用页面上的"导出 PNG 图标"按钮（需要浏览器支持）

## 图标尺寸要求

### 微信小程序图标尺寸规范

1. **小程序图标**（app.json 中配置）
   - 尺寸：1024×1024 像素
   - 格式：PNG
   - 大小：建议小于 200KB
   - 用途：小程序分享、搜索展示等

2. **TabBar 图标**（已配置在 app.json）
   - 尺寸：81×81 像素
   - 格式：PNG
   - 大小：建议小于 40KB
   - 用途：底部导航栏图标

## 配置步骤

### 1. 生成 PNG 图标

按照上述方法生成以下尺寸的 PNG 图标：
- `app-icon-1024.png` (1024×1024)
- `app-icon-81.png` (81×81) - 如果需要替换 TabBar 图标

### 2. 放置图标文件

将生成的 PNG 文件放置到：
```
miniprogram/images/
├── app-icon-1024.png  (小程序图标)
└── app-icon-81.png    (TabBar 图标，可选)
```

### 3. 在微信开发者工具中配置

1. 打开微信开发者工具
2. 点击"项目" → "设置"
3. 在"项目设置"中找到"小程序图标"
4. 上传 `app-icon-1024.png` 文件
5. 保存配置

### 4. 替换 TabBar 图标（可选）

如果需要使用新图标作为 TabBar 图标，可以：
1. 将 `app-icon-81.png` 重命名为对应的 TabBar 图标名称
2. 或修改 `app.json` 中的 `iconPath` 指向新图标

## 配色方案

图标使用的配色方案：
- **主渐变**：#6366F1 → #8B5CF6
- **主色调**：#6366F1
- **辅助色**：#8B5CF6
- **高光色**：#A78BFA
- **背景色**：白色（图标内部）

这些颜色与小程序整体设计保持一致。

## 注意事项

1. **图标大小**：确保 PNG 文件大小符合微信要求
2. **图标清晰度**：使用高分辨率导出，确保在小尺寸下也清晰可见
3. **图标圆角**：微信小程序会自动添加圆角，设计时无需手动添加
4. **图标背景**：建议使用透明背景或纯色背景
5. **图标测试**：上传后在不同设备上测试显示效果

## 自定义修改

如果需要修改图标设计：

1. 编辑 SVG 文件：`miniprogram/images/app-icon.svg`
2. 使用文本编辑器或 SVG 编辑工具打开
3. 修改颜色、形状、大小等
4. 保存后重新转换为 PNG

### 修改颜色示例

在 SVG 文件中找到渐变定义，修改颜色值：
```xml
<linearGradient id="mainGradient" ...>
  <stop offset="0%" style="stop-color:#6366F1;stop-opacity:1" />
  <stop offset="100%" style="stop-color:#8B5CF6;stop-opacity:1" />
</linearGradient>
```

将 `#6366F1` 和 `#8B5CF6` 替换为你想要的颜色。

---

**提示**：如果遇到图标显示问题，请检查文件路径、文件大小和格式是否符合微信小程序的要求。
